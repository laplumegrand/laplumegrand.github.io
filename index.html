Assignment 1

Some text, then a hyperlink about GIS info on Baton Rouge.

Assignment 2

Created a web app that gives a virtual tour of a few unique spots and sites in Chicago.

Assignment 3

Creating a web app that can show how population in the US changes over time, is one of the many ways that one can use GIS and web applications in conjunction with one another.

Assignment 4

Created a web app with ArcOnline to allow people to input information directly into a hosted layer.

Assignment 5

Created a web app that shows the locations and some spatial relationships between grocery stores, daycare centers, and bus stops in East Baton Rouge Parish.

Assignment 6

Created a web app that shows the change in where the 100 most populous cities in the U.S. are from 1798 - 2009. It also shows the rank and population of the city.
